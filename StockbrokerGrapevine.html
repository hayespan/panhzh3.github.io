第12次练习B题题解

题目链接：http://vjudge.net/contest/view.action?cid=47127#problem/B

题目大意：

这道题目就是说要向n（n<=100）个经纪人散步谣言，数据给出的是跟第i人有联系的人、第i人跟他有联系的人联系所需的时间t。问的是应该从哪个人开始散布谣言，时间才能最短，并且求这个最短时间。

输入：

　　多个测试案例，n==0时结束。

　　接下来共n行，第i行：

　　联系人数，pair(联系人，时间),pair(联系人，时间)...

思路:

利用floyd求任意两点的最小距离，然后找出起始点，使得该点到其它所有点的时间总和最小。输出该点，并输出与该点最远的点的距离。

特殊情况：可能对于某个测试样例，无论从哪一点开始，均无法完整到达其他所有点，也即谣言无法完全传播，那么无解，输出disjoint。判断方法为，最小时间总和大于无穷大（某个大数）。

算法步骤：

  1. 输入
  2. Floyd
  3. 找起始点
  4. 判断是否有解
  5. 输出

算法复杂度：

Floyd复杂度为$O(n^3)$

查找起始点复杂度为$O(n^2)$

查找最远点复杂度为$O(n)$

所以总的复杂度为$O(n^3)$

源程序：

#include 
#include 
using namespace std;

#define REP(i, n) for(int i=0; i> n && n!=0) {
        REP(i,n)
            REP(j,n)
                g[i+1][j+1] = i==j?0:M;
        REP(e,n) {
            int t;
            cin >> t;
            int v,w;
            REP(i,t) {
                cin >> v >> w;
                g[e+1][v] = w;
            }
        }
        floyd();
        int p,m = M;
        REP(i,n) {
            int sum=0;
            REP(j,n)
                sum += g[i+1][j+1];
            if(sum=M)
            cout << "disjoint" << endl;
        else {
            int t = 0;
            REP(i,n)
                t = t < g[p][i+1]?g[p][i+1]:t;
            cout << p << " " << t << endl;
        }
    }
    return 0;
}
